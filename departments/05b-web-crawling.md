## Recommended crawling stack for THIS company
| Purpose | Project |
|---|---|
| Internet/social research | Agent-Reach |
| Local crawling | Crawl4AI |
| AI web extraction | Firecrawl |
| Browser automation | Browser Use |
| Reliable browser | Playwright |
| Private search | SearXNG (33.7k★, AGPL) |

## Why
AI-native crawlers (Firecrawl/Crawl4AI) output clean Markdown/JSON and integrate with LLM + RAG
pipelines — far better than legacy scrapers for an agent company. Agent-Reach covers social intel.
