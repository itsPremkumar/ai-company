## How the company runs itself

```
Heartbeat ─► agents self-dispatch assigned issues
   │
   ├─ CTO/HoP/QA ──► product engineering (AVG), CI, release gates
   ├─ CMO/CFO ────► pricing, outreach drafts, content, revenue tracking
   ├─ COO ────────► delivery ops, SLA, client workflows
   └─ Watchdog ───► keeps server alive, re-auths, writes health report
```

All runs are inspectable: agents record work products and dispositions; the watchdog logs health.
The human steers strategy and owns the publishing accounts. Agents draft; the human ships.
