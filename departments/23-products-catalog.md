## 🔄 The build→sell loop (all in this stack)
```
AI Product Mgr (CrewAI) → picks product from research (Agent-Reach)
   → Coding dept (OpenCode/Hermes/OpenHands) builds it
   → QA (Hermes + langfuse) tests it
   → Deploys (Coolify + Docker)
   → Marketing (Mautic + Video-Gen) promotes
   → Sales/CRM (NocoBase/Medusa) closes
   → Finance (ERPNext) invoices
   → Support (Chatwoot) retains
```

## ⚠️ Reality check
- Coding agents (OpenCode/OpenHands) build **scaffolds + features**, not flawless production apps
  unaided — Hermes + human review still needed for shipping quality.
- Start with **1 product, prove revenue**, then expand. Don't build 10 at once.
- Client work funds SaaS until SaaS self-sustains.
